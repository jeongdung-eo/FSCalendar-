## 구현 기능
- [x] header 변경
- [x] 요일 순서 변경, 일요일만 날짜 색상 red로 변경 
- [x] 다음달, 이전달 버튼 생성 
- [x] 날짜 클릭 시 액션 없음
- [ ] 성취율에 따른 circle 색상
